## Screenshot Guidelines

### Taking Screenshots

1. **Clean State**: Start with a fresh app state or use sample data
2. **Example Content**: Use non-sensitive, example translation text
3. **Consistent Theme**: Use dark theme for most screenshots (more popular)
4. **Window Size**: Set window to consistent size before capturing
5. **OS Chrome**: Include window title bar and borders
6. **High DPI**: Take screenshots on high-resolution display if possible

### Sample Translation Content

Use these example translations for consistency:

- **English → Russian**: "Hello, world!" → "Привет, мир!"
- **Russian → English**: "Как дела?" → "How are you?"
- **English → German**: "Good morning" → "Guten Morgen"
- **French → English**: "Bonjour" → "Hello"

### Tools

Recommended screenshot tools:
- **Windows**: Snipping Tool, ShareX
- **macOS**: Command+Shift+4 (built-in)
- **Linux**: Flameshot, GNOME Screenshot

### Editing

- **Format**: PNG (lossless)
- **Optimization**: Use TinyPNG or similar to reduce file size
- **No Watermarks**: Keep screenshots clean
- **Cropping**: Remove unnecessary margins
- **Annotations**: Add if highlighting specific features
